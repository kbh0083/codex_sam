- Use the exact numeric value from "설정금액" as SUB and the exact numeric value from "해지금액" as RED for the same fund. Do not net them unless a dedicated net column exists.

- The manager column is "운용사명". Keep only Samsung-managed rows when the manager value indicates 삼성.
- Ignore subtotal and total rows such as rows whose fund code is an aggregate label ending with "합계".
- Rows whose effective amount is 0, "-", or blank are not transactions for that direction.
- Emit orders only from Samsung-managed rows whose fund identity and non-zero amount cell are explicitly visible.
- Do not invent extra fund rows, directions, or buckets that are not printed in the main table.
